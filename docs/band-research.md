# Band Artwork Research — Visual Reference Archive

**Purpose:** Reference archive of visual moves pulled from band artwork across the Pistoleros reference stack. This document is the *source material* — raw observations, categorised. The synthesised direction lives in `design-direction.md`.

**Method:** For each band, work through eras chronologically. For each era, capture the specific *visual moves* worth stealing — not "Maiden looks cool" but "Riggs uses hand-painted cartouches with hieroglyphic border treatments framing a central scene."

**Session log:** Started [DATE]. Working sessions documented at the bottom.

## 1. Reference Stack

Primary reference bands, in priority order for this project:

1. **The Damned** — first-wave UK punk going gothic
2. **Misfits** — horror-punk, graphic reduction, Crimson Ghost
3. **Iron Maiden** — painted theatricality, Riggs-era cartouches
4. **KISS** — comic-book bombast, poster theatricality, glam spectacle
5. **Hanoi Rocks** — glam-punk archetype, lipstick and leather
6. **The Cult** — gothic-arena-rock
7. **Killing Joke** — industrial gothic
8. **Ghost** — religious iconography weaponised (used sparingly)
